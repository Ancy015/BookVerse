# 📚 BookVerse — Bookstore Landing Page

A modern, fully responsive bookstore landing page built with pure HTML, CSS, and JavaScript.

## 🚀 Features

- **Sticky Navigation** with active link highlighting and mobile hamburger menu
- **Hero Section** with floating badges, animated book illustration, and stats counter
- **Features Section** with hover-animated cards
- **Popular Books Grid** with buy-now cart functionality and toast notifications
- **Category Cards** with ripple click effects
- **About Section** with mission & vision
- **Contact Section** with email, phone, address
- **Footer** with social icons and quick links
- **Back to Top** button
- **Scroll animations** (Intersection Observer API)
- **Fully responsive**: Mobile (320px+), Tablet (768px+), Desktop (1024px+)

## 🛠 How to Run

Simply open `index.html` in any modern browser — no build step required.

## ✅ Technical Stack

- **HTML5** — Semantic tags (`header`, `nav`, `main`, `section`, `footer`, `article`)
- **CSS3** — Custom properties, Flexbox, Grid, animations, media queries
- **Vanilla JS** — IntersectionObserver, DOM manipulation, event delegation
- **Google Fonts** — Inter (body) + Playfair Display (headings)
